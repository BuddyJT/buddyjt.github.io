# SecraCloud GitHub Pages Site

This is a static GitHub Pages-ready site for SecraCloud. It uses the attached
private-cloud visual system assets with SecraCloud public brand text.

## Files

- `index.html`: home page with Home, Privacy Policy, and Sign In navigation
- `privacy.html`: SecraCloud Privacy Policy & Disclaimer
- `404.html`: GitHub Pages-friendly not found page
- `styles.css`: shared site styling
- `assets/`: copied logo mark, favicon, and background image
- `.nojekyll`: disables Jekyll processing on GitHub Pages

## Publish With GitHub Pages

1. Create a new GitHub repository, or open the repository you want to use.
2. Upload the contents of this folder to the repository root.
3. In GitHub, go to Settings, then Pages.
4. Set the source to deploy from the `main` branch and the root folder.
5. Save the setting and wait for GitHub Pages to publish the site.

If you want the site at a custom domain, add a `CNAME` file containing the exact
domain after DNS is ready.

## Link Targets

- Sign In links point to `https://cloud.secracloud.com`.
- Privacy links point to `privacy.html`.

The policy copy is written for transparency during development. Have it reviewed
before using it as a production legal document.
